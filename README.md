# 👥 HR Analytics Dashboard

## 📌 Overview

This project presents an interactive **HR Analytics Dashboard** developed using **Power BI**. The dashboard analyzes employee data to provide insights into workforce performance, employee satisfaction, attrition, and departmental trends through interactive visualizations and KPIs.

---

## 🚀 Features

- Interactive KPI Cards
- Employee Performance Analysis
- Attrition Analysis
- Department-wise Workforce Insights
- Job Satisfaction Analysis
- Overall Employee Score using DAX
- Dynamic Filters & Slicers

---

## 🛠️ Tech Stack

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📈 Dashboard Insights

- Monitor employee performance across departments.
- Identify high-performing employees using an Overall Employee Score.
- Analyze attrition and workforce distribution.
- Evaluate employee satisfaction and performance ratings.
- Support HR decision-making through interactive dashboards and business insights.
